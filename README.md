# ✨ 功能说明
## 🎯 免Vip功能
1. 排行榜-热播, 免Vip查看
2. 排行榜-Top250, 免Vip查看
3. FC2视频页面调整, 免VIP查看
4. 所有评论信息, 免Vip查看
5. 相关清单, 免Vip查看

## 📺 视频列表页
1. 左键弹窗打开, ctrl+左键新窗口打开, 右键为屏蔽视频
2. 鉴定记录: 可对已标记的数据进行查看; 支持对db个人页面想看|看过数据导入到脚本数据中来进行管理
3. 新作品检测: 对已收藏演员进行统一管理, 同时展示最新作品数量
4. 演员黑名单: 对加入黑名单的演员进行统一管理, 可查看被屏蔽的番号总数量, 男女演员数, 检测状态等信息... 
5. 设置: 阿里云盘,WebDav在线备份、导入导出数据、自定义页面布局、定时任务配置、快捷键配置、缓存清理等...
6. 支持瀑布流方式加载页面数据、标题翻译、悬浮大图等功能

## 🎬 视频详情页
1. 可对当前视频标记为屏蔽(快捷键a,有询问窗口,需按2下,防误触), 收藏(快捷键s), 加入已下载
2. 评论区可展示更多评论信息; 选中评论区文本, 点击右键, 可加入评论区关键词过滤屏蔽
3. 标题关键词过滤: 顶部大标题位置, 选中文本, 点击右键, 可加入标题关键词过滤屏蔽
4. 支持SubtitleCat、迅雷字幕文件搜索
5. 支持JavTrailers、MiSSAV、Jable等第三方资源检测及快速跳转
6. 磁力过滤功能, 隐藏劣质磁力, 优先展示中文磁力, 高亮4K磁力 
7. 磁力搜索功能, 支持U9A9、U3C3、Sukebei等磁力网站快捷查询
8. 支持115网盘一键离线转存, 支持115资源匹配检测
9. 女优信息获取, 展示年龄、身高、体重、三围、罩杯等信息
10. 更多更高画质的预览视频解析, 快捷键z, 每次按下快进5秒 
11. 支持长缩略图查看

## 💃 演员页
1. 自动同步已收藏的演员信息, 支持新作品检测功能
2. 支持屏蔽演员,加入黑名单,自动检测屏蔽后续作品
